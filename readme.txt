== Description ==

Plugin para buscar palavras no html da página estática.

== Installation ==

1. Upload `search-html.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('search-html_hook'); ?>` in your templates
1. Use [shortSH]
